# App Web Órdenes

Frontend web para sistema de gestión de órdenes conectado a API REST desarrollada en Flask.

## Tecnologías
- HTML
- CSS
- JavaScript
- Fetch API

## Funcionalidades
- Gestión de órdenes
- Registro de clientes
- Control de pagos
- Estados de pedidos
- Interfaz operativa para locales comerciales
- Integración con backend API REST

## Estado del proyecto
Proyecto en desarrollo activo, actualmente en uso en locales reales, en proceso de mejora continua e implementación de buenas prácticas de frontend.

## Proyecto personal
Sistema desarrollado como proyecto personal productivo para operación comercial real.
